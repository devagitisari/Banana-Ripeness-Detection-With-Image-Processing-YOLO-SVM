# Banana Ripeness Detection üçå
This project focuses on detecting banana ripeness using image processing and machine learning techniques. YOLO is used for banana object detection, while SVM is applied for ripeness classification.

## Features
- Banana detection using YOLO
- Ripeness classification (unripe, ripe, overripe, rotten)
- ROI-based image processing
- Feature scaling and PCA integration

## Tech Stack
- Python
- OpenCV
- YOLO
- Scikit-learn

## Project Structure
- banana_detection.py : main program
- test.py             : testing script
- svm_model.pkl       : trained SVM model
- scaler.pkl          : feature scaler
- pca.pkl             : PCA model

## Model & Dataset
YOLO model weights (`best.pt`) and image dataset are not included in this repository due to file size limitations.
